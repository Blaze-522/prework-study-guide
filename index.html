HTML

  • The structure for most things front end.
  • Represents most text based front end product.
  • Goes hand in hand with CSS to make the front end of a product visually appealing.
  • Most all websites or apps use this for its interface.

CSS

  • Used to effect things like font size color and most of the cosmetic aspects that HTML lacks in.
  • HTML's right hand.
  • Can do pretty much anything to help make a product look however the developer intends.

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git is used to help keep our repository in check and individualize how our pc connects to it's service with other users.

JavaScript

  • Javascript is all of the fancy details of a website.
  • These things can include integrating payment options or other interactable features.
  • Javascript is very integral to things like structuring API's or even dfeveloping games.
  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
